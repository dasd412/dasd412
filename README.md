
# 자기 소개

- 시드 레벨 스타트업의 2호 직원으로서 백엔드 시스템의 70%를 밑바닥부터 설계 및 개발하였고,  TIPS 5억 투자 유치에 기여하였습니다.
- 현재 문제 상황과 일정을 고려하여 기술을 적용합니다.
- 단순히 기능만 구현하는 것에 그치지 않고, API 버전 호환성 등 전체 시스템에 미칠 수 있는 영향까지 고려합니다.
- API 문서, 도메인 규칙, 용어 사전 등 사내에 필요한 문서들을 정리하여 실무자들 간의 커뮤니케이션을 돕는 개발자입니다.
